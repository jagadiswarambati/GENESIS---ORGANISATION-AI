# Technology Stack

| Area                      | Choice                              | Role                                                |
| ------------------------- | ----------------------------------- | --------------------------------------------------- |
| Web application           | Next.js 15, React 19, TypeScript    | App Router-based product surface                    |
| Styling                   | Tailwind CSS, shadcn/ui conventions | Consistent, accessible UI foundation                |
| Interaction               | Framer Motion, React Flow, Lucide   | Future motion, organization graphs, and iconography |
| API                       | FastAPI, Pydantic                   | Typed backend boundary                              |
| Data                      | PostgreSQL, SQLAlchemy, Alembic     | Durable data and controlled migrations              |
| Semantic retrieval        | pgvector                            | Future vector similarity support                    |
| Background infrastructure | Redis                               | Future queue and coordination capability            |
| Local environment         | Docker Compose, uv                  | Repeatable developer setup                          |

Milestone 1 installs the foundations only. No API routes, user interface,
business logic, authentication, or AI integration is included.
